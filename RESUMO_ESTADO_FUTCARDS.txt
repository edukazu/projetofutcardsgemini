=== 1. STATUS GERAL ===
O jogo inicializa, carrega os times, executa um sorteio de "Cara ou Coroa" em tela cheia (3D) e realiza o pontapé inicial (Kickoff) respeitando a regra de passar para um Meio-Campista (MID). O visual mudou para um estilo "Diorama Tático" (Vertical).

=== 2. MUDANÇAS VISUAIS IMPORTANTES (Conceito Diorama) ===
* Orientação: O jogo agora usa formação VERTICAL (Cluster column) para simular as linhas do campo.
* Cartas: Não usamos mais tokens. Usamos as cartas completas (Full Art) reduzidas no campo.
* Campo: Altura aumentada drasticamente (85vh) para caber 5 cartas em pé. Largura reduzida (82%) para dar respiro lateral.
* Gol: Aumentado para 360px de altura para alinhar com o Goleiro.

=== 3. LÓGICA DE CÂMERA ===
* Sistema: Pan & Zoom via classes CSS (`cam-pos-X`, `cam-zoom-X`).
* Comportamento: A câmera foca na zona onde está a bola.
* Bug Atual: O "Pan" pode estar invertido dependendo do lado do campo.
